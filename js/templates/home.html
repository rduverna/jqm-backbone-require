This is a single page boilerplate template that you can copy to build your first jQuery Mobile page.

This project uses the following libraries :

  • JQuery
  • JQuery Mobile for the Mobile UI Framework
  • Underscore for the templating system
  • RequireJS for Asynchronous Module Definition
  • Backbone for code organization (MVC) & a better routing system

Important things

The JQuery mobile default routing system is disabled and replaced by a Backbone router. This allow to pass and easily retrieve arguments between pages. You can also use the Azicchetti router/controller for JQuery mobile.
